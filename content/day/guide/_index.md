## 📑 網站使用說明

歡迎來到「奇蹟 30」！為了讓您有更順暢的學習體驗，請參考以下網站使用說明。

## 🚀 自動前往今日課程

當您開啟本站首頁 [{{< site-url>}}]({{< site-url>}}) 時，系統會自動為您導向到今天的日期所對應的課程頁面。您無需手動尋找，即可開始當日的學習。

## 📖 自由選擇課程

如果您想複習之前的課程，或預覽未來的內容，可以透過以下方式導航：

- 💻 **電腦版：** 請使用左側的課程目錄，點擊您想前往的日期。
- 📱 **手機版：** 請點擊左上角的「選單」圖示，即可展開課程目錄，自由選擇不同日期的課程。

## 🎧 音檔播放功能

每個課程頁面都附有相關的音檔，點擊播放器即可開始或暫停播放，建議您在安靜的環境下聆聽，以達到最好的學習效果。

## 📱 安裝到您的裝置 (PWA)

本站支援 PWA (Progressive Web App)，您可將本站安裝到手機或電腦上，享受如原生 App 般的沉浸式體驗與離線使用功能。

### 🤖 Android 設備安裝指南

#### 使用 Chrome 瀏覽器

1. **開啟網站**
   - 在 Chrome 瀏覽器中開啟 `{{< site-url >}}`

2. **尋找安裝提示**
   - 當網站載入完成後，瀏覽器下方會出現「新增到主螢幕」的提示橫幅
   - 或者點擊右上角選單 (⋮) → 選擇「新增到主螢幕」

3. **確認安裝**
   - 點擊「新增」或「安裝」按鈕
   - 系統會詢問是否要在主螢幕建立捷徑，點擊「新增」

4. **完成安裝**
   - 應用程式圖示會出現在您的主螢幕上
   - 點擊圖示即可像原生 App 一樣開啟「奇蹟 30」

#### 使用其他瀏覽器

如果您使用 Firefox 或其他瀏覽器：
- 找到瀏覽器選單中的「新增到主螢幕」或「安裝應用程式」選項
- 步驟與 Chrome 類似

### 🍎 iOS 設備安裝指南

#### 使用 🧭 Safari 瀏覽器

1. **開啟網站**
   - 在 🧭 Safari 瀏覽器 中開啟 `{{< site-url >}}`

2. **開啟分享選單**
   - 點擊底部工具列的「分享」按鈕 (正方形加箭頭圖示)

3. **新增到主螢幕**
   - 在分享選單中向下滑動
   - 找到並點擊「新增到主螢幕」選項

4. **自訂名稱**
   - 您可以編輯應用程式名稱
   - 點擊右上角的「新增」

5. **完成安裝**
   - 應用程式圖示會出現在您的主螢幕上
   - 點擊圖示即可全螢幕模式開啟網站

#### 注意事項
- iOS 目前僅支援 🧭 Safari 瀏覽器安裝 PWA
- 其他瀏覽器 (Chrome、Firefox 等) 暫時無法安裝 PWA

### 💻 macOS 設備安裝指南

#### 使用 🧭 Safari 瀏覽器

1. **開啟網站**
   - 在 🧭 Safari 瀏覽器 中開啟 `{{< site-url >}}`

2. **新增到 Dock**
   - 點擊頂部選單「檔案」→「新增到 Dock」
   - 或使用快捷鍵 `⌘ + Shift + D`

   - 點擊右上角的「新增」

   - 點擊圖示即可全螢幕模式開啟「奇蹟 30」

### 🛠️ 疑難排解

#### 找不到安裝選項？

**🤖 Android/Chrome:**
- 確保您使用的是最新版本的 Chrome
- 嘗試重新整理頁面
- 檢查是否已經安裝過 PWA

**🍎 iOS/🧭 Safari 瀏覽器:**
- 確保您使用的是 🧭 Safari 瀏覽器
- iOS 版本需要 11.3 或更新
- 嘗試關閉並重新開啟 🧭 Safari 瀏覽器

**💻 macOS:**
- 🧭 Safari 瀏覽器 版本需要 14 或更新
- 確保您使用的是 🧭 Safari 瀏覽器

#### 安裝後找不到應用程式圖示？

**🤖 Android:**
- 檢查主螢幕的所有頁面
- 查看應用程式抽屜
- 在設定中查看已安裝的應用程式

**🍎 iOS:**
- 檢查主螢幕的所有頁面
- 使用 Spotlight 搜尋「奇蹟 30」
- 檢查「App 資料庫」

**💻 macOS:**
- 查看應用程式資料夾
- 使用 Spotlight 搜尋
- 檢查 Dock 中是否已新增

#### 需要移除 PWA？

**🤖 Android:**
- 長按應用程式圖示 → 選擇「解除安裝」
- 或到設定 → 應用程式 → 找到「奇蹟 30」→ 解除安裝

**🍎 iOS:**
- 長按應用程式圖示 → 選擇「移除 App」
- 確認刪除

**💻 macOS:**
- 將應用程式從應用程式資料夾拖到垃圾桶
- 或右鍵點擊 → 「移到垃圾桶」
